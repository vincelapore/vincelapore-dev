# vincelapore.dev

Portfolio site for Vince Lapore — developer specialising in web apps and digital products. Built with Next.js (App Router) and TypeScript.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy

Deploy on [Vercel](https://vercel.com) (or any Node host). Connect this repo and use the default Next.js build settings.
